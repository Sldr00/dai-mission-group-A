## Contents
- `group_a_proposal.ipynb` — final notebook (fully executed)
- `requirements.txt` — required Python packages
- `group_a_presentation_slides.pdf` — presentation slides
- `Glassdoor Gender Pay Gap.csv` — dataset (Kaggle, CC0)

### How to run

pip install -r requirements.txt

Then open group_a_proposal.ipynb and run all cells.

### Summary

We estimate the causal effect of gender on base salary (propensity score
stratification), compare predictive models (Linear, Ridge, Lasso, Decision
Tree, Random Forest), and use K-Means clustering to explore heterogeneous
treatment effects across employee segments.
